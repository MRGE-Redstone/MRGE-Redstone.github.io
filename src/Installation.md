# Installation

> [!WARNING]
> This guide only covers Linux Systems.

## Python Version

MCpypack requires Python 3.14 or higher.  
Run:
```bash
python --version
```

It should at least show:
```
Python 3.14.0
```

## Virtual Environment

The next step is to create a new virtual environment in the directory of your
project.

```bash
python -m venv venv
```

Then activate it.

```bash
source venv/bin/activate
```

## Install MCpypack

Now you can install MCpypack via pip with the following command:

```bash
pip install MCpypack
```
