# Adding Recipes

In this chapter we will discover how to create custom recipes using MCpypack.

> [!WARNING]
> Although creating recipes is quite easy to understand we won't go into detail about specific recipe types and what they do. To get a better understanding of this you can inform yourself using [other sources](https://minecraft.wiki/w/Recipe_(Java_Edition)).

> [!IMPORTANT]
> This chapter uses features which recipes depend on but which are not explained in this chapter.
> These featues include:
> - Item and Tag
> - ItemStack
